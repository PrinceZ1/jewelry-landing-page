# Jewelry — Landing page

A single-page marketing site for a premium jewelry brand, built with Vite and React.

## Stack

React 19 · TypeScript (strict) · Vite 8 · CSS Modules · Google Fonts (Playfair Display, Montserrat, Cormorant Garamond) · inline SVG

## Commands


| Command           | Description                            |
| ----------------- | -------------------------------------- |
| `npm install`     | Install dependencies                   |
| `npm run dev`     | Dev server (Vite default port)         |
| `npm run build`   | `tsc -b` + production build to `dist/` |
| `npm run preview` | Preview production build locally       |
| `npm run lint`    | Run ESLint                             |


## Deployment

Run `npm run build`, then deploy the `**dist/**` folder to any static host (Vercel, Netlify, S3, etc.).
